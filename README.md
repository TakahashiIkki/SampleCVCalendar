# SampleCVCalendar

[Mozharovsky/CVCalendar](https://github.com/Mozharovsky/CVCalendar) の実装サンプルです。

CVCalendarライブラリを使って、カレンダーを表示しています。

現在、EventKitを使って、カレンダーからの情報取得を実装している所です。
